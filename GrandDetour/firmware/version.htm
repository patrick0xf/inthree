Firmware Release Information

Version 2.8.0

The ICM now supports remote access from www.myEcoConcierge.com. This website makes it very simple to enable remote access to ICMs, with no need to use static IP addressing, forward router ports or compromise the local firewall.


Note: Reset the ICM to factory defaults after the upgrade to enable all the new features.

Upgrade Information

Upgrade Instructions

  • Enter http://myhome/setup in the address bar of the web browser.
  • Click Advanced in the navigation bar on the left side of the screen, then click Upgrade.
  • Select the Node to upgrade from the list, then click AutoUpgrade.
  • At the end of the upgrade process, select Reset to factory defaults to enable all the new features. This will require you to reconfigure the ICM.
  • Make sure you upgrade all the ICMs on the network to the same firmware version.
Note: If the upgrade fails, verify your Internet connection. Wait 5 minutes, and then restart the ICM. Try the upgrade process again.
